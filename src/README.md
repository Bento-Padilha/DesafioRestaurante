# Sistema de Controle de Pedidos de Restaurante

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos.

## Descrição

O sistema simula o controle de pedidos de um restaurante, permitindo:

- Cadastro de cliente
- Registro de pedidos
- Cálculo de subtotal
- Cálculo de taxa de serviço
- Aplicação de desconto
- Exibição do valor final da conta

## Estrutura

O projeto foi dividido em classes:

- Cliente → dados do cliente e manipulação de String
- Pedido → informações dos pedidos e cálculo do total
- ContaRestaurante → regras de negócio e cálculos finais
- Main → execução do programa

## Tecnologias

- Java
- Programação Orientada a Objetos (POO)

## Execução

Basta executar a classe `Main.java`.
